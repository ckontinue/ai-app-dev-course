# Learning Record 0001 — AI 应用开发全景图

## 核心收获

- AI 应用工程师 = 用模型的人，不是训练模型的人
- LLM 是大语言模型（GPT、Claude 等），通过 API 调用
- RAG = 检索增强生成，给 AI 外挂知识库
- Agent = AI + 工具调用能力，能动手做事
- 技术栈五层：数据层 → 模型层 → 模型服务层 → 应用逻辑层 → 前端界面层

## 关键类比

| 概念 | 类比 |
|------|------|
| LLM | 刚毕业的大学生，知识渊博但不知道你的业务 |
| RAG | 给他一本你的内部手册 |
| Agent | 给他钥匙和权限，他能帮你办业务 |
| Function Calling | 告诉他"抽屉里有印章，需要就盖" |
| Prompt | 你怎么跟他说清楚要干什么 |
