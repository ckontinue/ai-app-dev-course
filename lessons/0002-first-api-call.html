📡 第一次调用 LLM API

Lesson 0002 · 2026-06-10 · 动手 10 分钟

你要干什么

写一段 Python 代码，给 Claude 发一句话，收它的回复。
就这么简单。这是所有 AI 应用的基础——先会对话，再谈其他。

三行代码搞定

import anthropic                             # 1. 导入 SDK

client = anthropic.Anthropic()              # 2. 创建客户端（自动读你的 API_KEY）

response = client.messages.create(          # 3. 发请求
    model="claude-sonnet-4-6",
    max_tokens=200,
    messages=[{"role": "user", "content": "用一句话介绍什么是 RAG"}]
)

print(response.content[0].text)             # 4. 打印回复

拆解每一行

代码白话解释
import anthropic 引入 Anthropic 的 Python 包，相当于"师傅我要用你的工具"
client = anthropic.Anthropic() 创建一个客户端。它会自动从环境变量 ANTHROPIC_API_KEY 里读你的密钥
model="claude-sonnet-4-6" 选模型。Sonnet 性价比最高，日常开发够用
max_tokens=200 限制回复长度。token ≈ 1.5 个中文字，200 约 300 字
messages=[...] 对话内容。只有一条用户消息，最简单的单轮对话
response.content[0].text 从返回包里取出文本内容。API 返回的是结构化数据，.text 才是字

API 请求的完整结构

你发出去的 JSON（SDK 帮你拼的，不用手写）：

{
  "model": "claude-sonnet-4-6",
  "max_tokens": 200,
  "messages": [
    {"role": "user", "content": "用一句话介绍什么是 RAG"}
  ]
}

服务器返回的 JSON：

{
  "id": "msg_01Xxxxx...",
  "model": "claude-sonnet-4-6",
  "role": "assistant",
  "content": [
    {"type": "text", "text": "RAG（检索增强生成）是..."}
  ],
  "usage": {"input_tokens": 15, "output_tokens": 42}
}

三个最重要的概念

1. model（模型） — 选哪个大脑。Sonnet 性价比，Opus 最强，Haiku 最快最便宜

2. messages（消息列表） — 对话历史。每次都要把之前的对话全发过去，因为模型本身"无记忆"

3. tokens（令牌） — 计费单位。中文约 1.5 字 = 1 token，输入输出都算钱
